# Tamil Nadu TANSEED 6.0

## Overview
Tamil Nadu's Startup Seed Grant Fund, now in its 6th edition. Offers **₹10 Lakh** (disbursed in milestone-based tranches). Now includes a **"Green Track"** specifically for climate-tech and sustainability ventures.

## Eligibility
- **Stage**: Pre-seed to seed
- **Sectors**: All sectors (Green Track for climate-tech)
- **Entity Type**: Tamil Nadu-based startups
- **Geography**: Tamil Nadu

## Funding Details

| Parameter | Details |
|-----------|---------|
| Amount | ₹10 Lakh |
| Disbursement | Milestone-based tranches |
| Special Track | Green Track (climate/sustainability) |
| Equity | None |
