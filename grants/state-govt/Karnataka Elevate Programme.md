# Karnataka Elevate Programme

## Overview
India's most robust state-level grant mechanism. Karnataka's **Elevate** programme has funded over **1,000 startups** without taking equity — a model for other states. Part of Karnataka's Startup Policy 2025-30.

## Eligibility
- **Stage**: Pre-seed to seed
- **Sectors**: All sectors
- **Entity Type**: Karnataka-based startups
- **Geography**: Karnataka

## Funding Details

| Parameter | Details |
|-----------|---------|
| Amount | Up to ₹50 Lakh |
| Type | Grant-in-aid (no equity) |
| Startups Funded | 1,000+ |
| Equity | None |
