# MeitY GENESIS

## Overview
GENESIS (Gen-Next Support for Innovative Startups) offers up to **₹10 lakh** in non-refundable grants for early-stage technology startups, including AI ventures, with mentorship from government-recognized incubators. Run by [[MeitY]].

## Eligibility
- **Stage**: Ideation to pre-seed
- **Sectors**: Technology, AI/ML, electronics, IoT
- **Entity Type**: Early-stage tech startups
- **Geography**: Pan-India, through MeitY-recognized incubators

## Funding Details

| Parameter | Details |
|-----------|---------|
| Amount | Up to ₹10 Lakh |
| Type | Non-refundable grant |
| Equity | None |
| Additional | Mentorship from govt-recognized incubators |
