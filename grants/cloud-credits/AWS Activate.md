# AWS Activate

## Overview
AWS credits program for startups. Multiple tiers from self-serve ($1K) to portfolio ($100K for accelerator/VC-backed startups). **DPIIT Startup India beneficiaries** can access Activate credits directly.

## Tiers
- **Founders**: Up to $1,000 in credits (self-serve)
- **Portfolio**: Up to $100,000 in credits (via accelerator/VC)
- **DPIIT Track**: Available to Startup India registered startups

## Additional Benefits
- AWS technical support
- Training credits
- Architecture guidance
- Startup-specific solutions
