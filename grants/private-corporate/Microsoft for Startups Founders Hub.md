# Microsoft for Startups Founders Hub

## Overview
Microsoft's startup support program providing **Azure cloud credits up to $150K**, software licenses, technical guidance, and enterprise access. Open to startups without investor backing (basic path) or with investor/accelerator backing (advanced path). **No equity taken.**

## Benefits
- Azure credits up to $150K
- GitHub Enterprise, Visual Studio Enterprise
- Microsoft 365 seats
- Technical mentorship
- Enterprise customer introductions
