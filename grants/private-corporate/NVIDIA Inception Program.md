# NVIDIA Inception Program

## Overview
NVIDIA's global startup program. Members get GPU credits, co-marketing support, preferred pricing on NVIDIA hardware, technical workshops, and access to NVIDIA's Deep Learning Institute. Inception members can also access **up to $100K in AWS credits** through the NVIDIA-AWS partnership.

Key partner in India's [[IndiaAI Mission]], with 20,000 GPUs deployed.

## Benefits
- GPU cloud credits and hardware discounts
- Co-marketing and brand visibility
- NVIDIA Deep Learning Institute access
- $100K AWS Activate credits (via partnership)
- Investor connect programs
